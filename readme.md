# Complexidade Computacional e Criptografia

Projeto acadêmico sobre RSA, fatoração inteira e complexidade computacional.

## Funcionalidades

- Implementação didática do RSA
- Geração de chaves
- Cifração e decifração
- Ataque por fatoração
- Benchmarks experimentais
- Visualização gráfica

## Execução

### Instalar dependências

pip install -r requirements.txt

### Executar RSA didático

python rsa_didatico.py

### Executar benchmark RSA

python benchmark.py

### Executar benchmark de fatoração

python benchmark_fatoracao.py
